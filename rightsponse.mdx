# Rightsponse - AI Writing Assistant

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/fadilsflow/rightsponse)

Transform your communication with AI-powered writing assistance. Perfect for emails, messages, responses, and AI prompts. Built with Next.js and Gemini API, it offers customizable language and tone options for better communication. Features include real-time text generation, multiple response formats, voice input capabilities, and a clean, modern UI powered by shadcn/ui.

## Fitur Utama

- **Berbagai Jenis Konten**
  - Peningkatan pesan chat
  - Format email profesional
  - Balasan komentar yang tepat
  - Peningkatan prompt AI
- **Dukungan Bahasa**

  - Bahasa Indonesia
  - Bahasa Inggris
  - Pilihan nada tulisan (profesional, santai, ramah)

- **Input Suara**

  - Fitur voice-to-text dalam Bahasa Indonesia
  - Tombol rekam dan stop yang mudah
  - Konversi suara ke teks secara instan

- **Antarmuka Modern**
  - Desain responsif untuk semua perangkat
  - Tab terpisah untuk setiap jenis konten
  - Penyimpanan state untuk setiap tab
  - Tombol salin cepat

## Teknologi yang Digunakan

- **Next.js 15.3.3**: Framework React untuk produksi
- **Gemini API**: Model bahasa AI canggih dari Google
- **TypeScript**: Untuk kode yang aman dan terpercaya
- **Tailwind CSS**: Framework CSS yang fleksibel
- **shadcn/ui**: Komponen UI berkualitas tinggi
- **Web Speech API**: Untuk fitur input suara

## Cara Memulai

Untuk petunjuk setup lengkap, kunjungi [Tutorial Lengkap](https://fadils.xyz/blog/posts/membuat-ai-rewriter-dengan-gemini-api-dan-Nextjs).

```bash
git clone https://github.com/fadilsflow/rightsponse
cd rightsponse
npm install
```

Dapatkan API key dari [Google AI Studio](https://aistudio.google.com/apikey) dan atur di file ".env":

```env
GEMINI_API_KEY=your_api_key_here
```

Jalankan server development:

```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
